Projet CLI :

Nous avons programmé et testé notre CLI sur Windows.
Le programme traduit donc la commande pour tuer un processus bing[-k] en commande de terminal Windows.

Les commandes que nous avons réussi à faire sont :

- bing[-k] <processId> qui tue le processus dont l'utilisateur entre le PID
- lp qui liste les processus en cours en les numérotant à partir de 1
- CTRL + P qui permet de quitter le programme, il ne faut pas faire cette commande dans VS Code mais dans un terminal. En effet CTRL + P est une autre commande sur VS Code.
- exécution au premier plan d'un programme du système d'exploitation en indiquant son chemin d'accès (absolu ou relatif : / ou ..), ou qu'il soit dans une variable PATH récupérée de l'environnement shell de l'utilisateur au démarrage.
- ! qui après un nom de programme permet de l'exécuter en tache de fond
